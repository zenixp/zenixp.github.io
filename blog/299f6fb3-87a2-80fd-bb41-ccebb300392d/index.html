  • 已安装 Docker 与 Docker Compose（参考官方文档）

# 踩坑记录

  • 踩坑 1：端口必须一致
    • 映射端口、容器内部端口与 external_url 中端口需一致。
    • 避免使用 8080，会冲突。
  • 踩坑 2：Windows 挂载权限问题
    • 宿主机为 Windows 时，推荐使用 “命名卷” 挂载，避免 Runner 上传 Artifacts 时出现 “目录无权限” 错误。参考：
