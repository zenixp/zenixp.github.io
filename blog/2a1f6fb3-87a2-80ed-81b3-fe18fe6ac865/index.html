# 前言

在学习 k8s 时，打包应用镜像，需要搭建 docker 私有仓库，网路上主流都是用 harbor，但 harbor 部署时占用了太多资源，对于使用过 maven 私有库的人应该不陌生的 nexus 就可以拿来作为 docker 私有仓库来使用。

# 配置

第一次登录默认账号是 admin，默认密码在./nexus-data/admin.password

启动后默认已经有 maven 和 nuget 的私库配置，但是没有 docker 的，需要我们自己创建

进入菜单 Settings > Repository > Repositories，点击 Create repository 创建仓库，类型选择 docker (hosted)

Name 这里设置成 docker-hosted 或者自己写一个

Allow anonymous docker pul 勾起来，其他默认配置即可
