db-query 是一个强大的 Clawdbot 技能，用于查询项目数据库并自动管理 SSH 隧道连接。

# 主要功能

  • 自动管理 SSH 隧道生命周期
  • 支持多个数据库配置
  • 查询前自动建立隧道，查询后自动关闭

# 使用方法

db-query 技能通过配置文件管理多个数据库连接，特别适合需要通过 SSH 隧道访问的数据库环境。

# 高级功能

  • 🔗 自动 SSH 隧道管理：查询前建立连接，完成后自动清理
  • 📊 结果格式化：支持 JSON、表格等多种输出格式
  • 🔐 安全连接：支持 SSH 密钥认证和密码认证
  • ⚡ 批量查询：一次执行多个 SQL 语句

# 适用场景

  • 开发环境需要访问生产数据库进行数据调试
  • CI/CD 流程中的数据库检查和数据迁移
  • 运维脚本中的数据库监控和统计
  • 多环境数据库的统一管理
